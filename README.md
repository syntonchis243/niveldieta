# NivelDieta

Aplicacion local para registrar calorias, proteina, carbohidratos y grasas por dia.

## Abrir

Abre `index.html` en el navegador. La informacion se guarda en `localStorage`, asi que no hace falta servidor ni instalacion.

## Incluye

- Selector desplegable de alimentos con macros por 100g.
- Registro por desayuno, comida, cena y snack.
- Alimentos personalizados.
- Objetivos diarios editables.
- Resumen de calorias y barras de macros.
